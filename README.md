# Student-Migration-Analysis-Dashboard
This project explores key patterns in student migration using an interactive Excel dashboard. The goal is to understand how age, gender, educational qualification, and destination countries influence migration trends and satisfaction with foreign education.

##Key Insights Covered
  Age Distribution of migrated students
  Gender Breakdown (Male vs Female)
  Destination Countries and their religion distribution
  Educational qualification at the time of migration (Degree, PG, Plus Two, Diploma)
  Migration timeline (Age by Year of Migration)
  Satisfaction level with foreign education

##Features of the Dashboard
  Clean visual layout designed for clarity
  Multiple chart types (bar, donut, line)
  Slicers for interactive filtering
  Easy-to-interpret insights for research and decision-making
  Built entirely using Microsoft Excel

##Tools Used
  Excel
  Pivot Charts
  Data Cleaning & Transformation
  Visualization Design
